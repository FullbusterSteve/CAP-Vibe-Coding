# Quality Check

This worklow ensures quality by running linting checks and tests 

## Step 1: Check prerequisites
Check that the ui5 mcp server is available

## Step 2: run the ui5 linter
Run the ui5 linter using the run_ui5_linter from the ui5 mcp server

## Step 3: Runtests
use the /test skill to run the tests

## Step 4: Perform security review

Use the security hardening skill to perform a security review
